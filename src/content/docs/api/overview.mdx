# API 概览

本项目使用 [tRPC](https://trpc.io/) 构建端到端类型安全的 API 层。所有端点通过 tRPC router 组织，支持自动类型推断，无需手动编写 API 客户端代码。

## 认证级别

| 级别 | 说明 |
|------|------|
| `public` | 无需认证，任何人可访问 |
| `protected` | 需要用户登录 |
| `admin` | 需要管理员角色 |
| `superAdmin` | 需要超级管理员角色 |

## auth - 认证路由

管理用户认证、会话和权限。

| 端点 | 类型 | 认证 | 说明 |
|------|------|------|------|
| `auth.checkLoginAttempts` | query | public | 检查登录尝试次数和账户锁定状态 |
| `auth.recordFailedLogin` | mutation | public | 记录登录失败尝试 |
| `auth.getCurrentUser` | query | public | 获取当前登录用户信息 |
| `auth.checkAuthStatus` | query | public | 检查认证状态 |
| `auth.updatePassword` | mutation | protected | 更新用户密码 |
| `auth.getSession` | query | public | 获取用户会话信息 |
| `auth.refreshSession` | mutation | protected | 刷新用户会话 |
| `auth.updateLastLogin` | mutation | protected | 更新最后登录时间 |
| `auth.getUserPermissions` | query | protected | 获取用户权限信息 |
| `auth.getUserSessions` | query | protected | 获取用户会话列表 |
| `auth.terminateSession` | mutation | protected | 终止指定会话 |
| `auth.terminateOtherSessions` | mutation | protected | 终止其他所有会话 |

## 使用示例

### 客户端调用

```typescript
import { trpc } from '@/lib/trpc/client'

// Query 示例
const { data: sessions } = trpc.aichat.listSessions.useQuery({
  limit: 10,
  page: 1,
})

// Mutation 示例
const createSession = trpc.aichat.createSession.useMutation()
await createSession.mutateAsync({ title: '新对话' })
```

### 服务端调用

```typescript
import { appRouter } from '@/server/root'
import { createContext } from '@/server/server'

const caller = appRouter.createCaller(await createContext())
const plans = await caller.payments.getMembershipPlans()
```
